## Descripción del Proyecto
Este proyecto es un portafolio digital interactivo diseñado para documentar y demostrar la aplicación práctica de los principios de la física mecánica en el desarrollo de software moderno. El sitio utiliza una estética minimalista y profesional para presentar de forma clara los conceptos evaluados durante el primer avance del curso.

## 🛠 Tecnologías Utilizadas
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Diseño responsivo y Dark Mode)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) (Simulaciones físicas interactivas)
- **Despliegue:** [Vercel](https://vercel.com/)

## 📚 Contenido Académico
El portafolio está estructurado en 4 módulos principales que cubren las semanas 2 a 5 del curso:

1. **Semana 2: Cinemática**
   - Movimiento en 3 Dimensiones (Niebla Volumétrica).
   - Movimiento Circular Uniforme (Sincronización de Discos HDD).
2. **Semana 3: Dinámica**
   - Fuerza Centrípeta (Control de Estabilidad en Drones).
   - Fricción Cinética (Resistencia Inercial en UI).
3. **Semana 4: Trabajo y Energía**
   - Trabajo Mecánico (Gestión de Carga en CPUs).
   - Conservación de la Energía (Eficiencia Energética en Móviles).
4. **Semana 5: Conservación y Colisiones**
   - Impulso (Orientación de Micro-satélites).
   - Colisiones Inelásticas (Simuladores de Impacto CAD).

## 🚀 Instalación y Ejecución Local

Si deseas ejecutar este proyecto localmente para desarrollo:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/TU_USUARIO/fisica-portfolio.git
   ```
2. **Instalar dependencias:**
   ```bash
   npm install
   ```
3. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```
4. **Abrir en el navegador:**
   Accede a `http://localhost:3000`.
